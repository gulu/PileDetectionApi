# 基桩完整性类别判定标准（声参量：声速、波幅、PSD）
## 完整性类别	测点声参量判定规则（声速Vi​、波幅Ai​）
Ⅰ 类	①所有测点 临界（无统计异常点）、临界（波幅无异常）②个别测点 临界＞临界 且 临界＞临界（深度方向、横向不连续，异常点数在 20 个点内）③多个测点 临界＞临界 或 临界＞临界（深度方向连续不超过 3 个点、横向不超过剖面数量的 50%）
Ⅱ 类	①个别测点 临界＞临界 且 临界＞临界（深度方向连续不超过 3 个点、横向不超过剖面数量的 50%）②多个测点 临界＜＜临界 或 ＜临界（深度方向不超过 3 个点、横向不超过剖面数量的 50%）
Ⅲ 类	①多个测点 ＜临界 或 临界＞（深度方向超过 4 个点、横向超过剖面数量的 50%）
Ⅳ 类	①多个测点 ＜临界 或 ＜临界（深度方向超过 4 个点、横向超过剖面数量的 50%）
